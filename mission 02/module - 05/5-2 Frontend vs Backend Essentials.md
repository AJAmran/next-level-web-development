# Frontend vs Backend Essentials

## Frontend

Frontend হলো website/app এর যে অংশ user দেখে এবং interact করে।

### Main Technologies

* HTML → structure
* CSS → design/style
* JavaScript → logic & interactivity

### Frontend কাজ

* UI design
* Responsive layout
* Animation
* API call
* User interaction handle করা

### Popular Frontend Tools

* React
* Next.js
* Tailwind CSS

---

## Backend

Backend হলো server-side logic যা user সরাসরি দেখে না।

### Backend কাজ

* Authentication
* Database handling
* API তৈরি
* Business logic
* Security

### Popular Backend Technologies

* Node.js
* Express.js
* Django
* Laravel

---

# Frontend vs Backend

| Frontend         | Backend                 |
| ---------------- | ----------------------- |
| User যা দেখে     | Hidden logic            |
| Browser এ run হয় | Server এ run হয়         |
| UI/UX handle করে | Data & logic handle করে |
| HTML/CSS/JS      | Node.js/Python/PHP      |
